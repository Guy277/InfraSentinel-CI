# InfraSentinel-CI
InfraSentinel CI est une plateforme intelligente de cyberdéfense qui permet aux entreprises de surveiller leurs systèmes et détecter précocement les comportements suspects grâce à l’intelligence artificielle.

La solution propose :

-une collecte et analyse des logs systèmes et réseau
-un moteur de détection d’anomalies basé sur l’intelligence artificielle
-un tableau de bord de sécurité permettant de visualiser les alertes
-un système de score de risque pour évaluer les menaces en temps réel
-une base de cybermenaces permettant le partage d’informations de sécurité entre entreprises
-L’objectif est de proposer une solution légère, accessible et adaptée aux réalités des entreprises africaines.
