# InfraSentinel CI

Plateforme intelligente de détection et prévention des cyberattaques pour les infrastructures critiques et les PME africaines.

## Contexte

Les cyberattaques ciblant les entreprises africaines sont en forte augmentation. 
Cependant, les solutions de cybersécurité avancées restent coûteuses et difficiles à déployer pour les PME.

InfraSentinel CI vise à démocratiser la cybersécurité grâce à une plateforme légère utilisant l'intelligence artificielle.

## Objectifs

- détecter les anomalies système et réseau
- identifier les comportements suspects
- générer des alertes en temps réel
- partager les informations de menace

## Fonctionnalités

- collecte de logs
- détection d'anomalies IA
- score de risque
- dashboard de sécurité
- partage de menaces

## Architecture

Logs → Analyse IA → Détection anomalies → Score de risque → Alertes → Dashboard

## Technologies

Python  
FastAPI  
Machine Learning  
PostgreSQL  
Dashboard Web

## Impact

InfraSentinel CI permet d'améliorer la cybersécurité des PME africaines et de renforcer la résilience des infrastructures critiques.
